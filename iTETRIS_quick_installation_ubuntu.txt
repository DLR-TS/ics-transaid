Step 1 - Install required libraries
$ sudo apt-get update
$ sudo apt-get install git autoconf automake libtool cmake libxerces-c-dev libfox-1.6-dev libgl1-mesa-dev libglu1-mesa-dev libgdal-dev libproj-dev libgeographic-dev python-pip
$ sudo pip install texttest

all following steps do not require sudo

--------------------------------
Step 2 - Get the source from git
$ git clone --recursive https://gitlab.imet.gr/hit/transaid
$ cd transaid
$ git checkout transaid-dev
$ git submodule update --init

The "original" repository containing the master branch for iTetris is located at https://gitlab.eurecom.fr/iTETRIS/iTETRIS-release

--------------------------------
Step 3 - Build SUMO
$ export SUMO_HOME=$PWD/share/sumo
$ mkdir sumo/build/cmake-build
$ cd sumo/build/cmake-build
$ cmake ../.. -DSUMO_UTILS=TRUE -DCMAKE_INSTALL_PREFIX=$PWD/../../..
$ make -j8
$ make install
$ cd ../../..

--------------------------------
Step 4 - Build NS-3.20
$ cd ns-3.20
$ ./waf configure --prefix=$PWD/..
$ ./waf -j8
$ ./waf install
$ cd ..

--------------------------------
Step 5 - Build ICS
$ cd iCS
$ autoreconf -i
$ ./configure --prefix=$PWD/..
$ make -j
$ make install
$ cd ..

Step 7 - Build testApp
$ cd iTETRIS-Applications/baseApp
$ autoreconf -i
$ ./configure
$ make -j
$ cd ../testApp
$ autoreconf -i
$ ./configure --prefix=$PWD/../..
$ make -j
$ make install
$ cd ../..

--------------------------------
Step 8 - Build the light-weight communication simulator
$ cd LightCommSimulator
$ autoreconf -i
$ ./configure --prefix=$PWD/..
$ make -j
$ make install
$ cd ..

When changing the baseApp remember to rebuild baseApp and afterwards every derived app (e.g. the testApp).

--------------------------------
Step 9 - Set required environment variables
$ export PATH=$PWD/bin:$PATH
$ export LD_LIBRARY_PATH=$PWD/lib
